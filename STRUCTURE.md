# Структура проєкту Constructor

Розділення зроблено для зручної роботи з окремими частинами без завантаження всього файлу (менше токенів при роботі з AI/редактором).

## Файли

| Файл | Призначення | ~рядків |
|------|-------------|--------|
| **index.html** | Розмітка сторінки, підключення стилів і скриптів | ~2645 |
| **css/main.css** | Змінні, базова верстка, компоненти, теми, анімації | ~1908 |
| **css/design-mode.css** | Стилі режиму дизайну (Design Mode Pro) | ~529 |
| **js/app.js** | Логіка конструктора: мапа, фільтри, генерація, формула | ~2629 |
| **js/design-mode.js** | Режим дизайну: перетягування вкладок, конфіг, збереження | ~1795 |
| **js/export-ui.js** | Експорт/прев’ю: toggleExportFields, handleSmartExecute | ~75 |
| **csv-layout.js** | Розклад полів CSV (зовнішній скрипт, як раніше) | — |

## Порядок підключення

1. **Head:** Bootstrap, іконки, Leaflet CSS, шрифти, Chart.js → `main.css`, `design-mode.css` → Sortable.js.
2. **Перед `</body>`:** Leaflet, Bootstrap bundle, `app.js`.
3. **Після `</body>`:** `csv-layout.js` → `design-mode.js`.

## Резервна копія

Повна однофайлова версія збережена як **index.html.backup** (усі стилі та скрипти всередині HTML).
